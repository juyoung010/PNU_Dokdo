GitHub Pages(정적) - 시트 연동은 웹앱에서만 가능

의견 수렴 안내

이 페이지는 GitHub Pages에 올라간 “정적 페이지”라서, 구글 시트 저장/조회 기능이 동작하지 않습니다.
의견 작성/수정/삭제/조회는 아래 버튼을 눌러 Google Apps Script 웹앱에서 진행해 주세요.

✅ 웹앱에서는 다음 기능이 가능합니다:
  • 의견 등록
  • 실시간 의견 목록 조회
  • 의견 수정 / 삭제
  • Google Sheets 자동 저장
메인으로 의견 작성하러 가기 (웹앱)

참고: GitHub Pages는 서버가 없어서 google.script.run이 동작하지 않습니다.
